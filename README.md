<h2>Project "DigiTools"</h2>

<p>This project contains AI related digital tools like AI Writing, Template Designer, Resume Builder, Social Media Optimizer, Social Media Content Kit etc. The main part of this project is product section functionality, like adding products to cart, removing product from cart etc.</p>

<h3>Technologies Used:</h3>

<p>• React.js</p>
<p>• Tailwind CSS</p>
<p>• JavaScript (ES6+)</p>
<p>• React-Toastify</p>
<p>• Lucide Icons</p>
<p>• JSON (for product data)</p>

<h3>Features:</h3>

<p>• By clicking the "Buy Now" button, the product is added to the cart, shows a success toast, the cart count increases, navbar cart count and the total amount in the cart section also increses.</p>

<p>• By clicking the "Remove" button on cart section, product is removed from cart, the cart count decreases, the navbar count and the amout in the cart section also decreases.</p>

<p>• By clicking the "Checkout" button, the cart is emptied.</p>
